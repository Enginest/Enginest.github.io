Deployment

Remarque

Vous devez mettre les 2 dossiers ci-dessous sur votre serveur puis configurer le serveur. Détails ici.

Note

Mettez votre version de TMS23 et POEM que vous utilisez pour le développement. (Pour avoir la même chose qu'en local, cela évitera des problèmes inutiles)

Attention

Assurez-vous d'avoir configuré le fichier "mobile.config" et "boot.config".
Détails ici.

  • AWS
Note

Si vous éteignez l'instance EC2 sur AWS, il faudra a nouveau télécharger le bureau à distance au prochain déploiement.

  • Démarer l'instance
  • Se connecter en RDP

Le mot de passe du bureau à distance est stocké dans le fichier "GDRIVE_PASSWORD.txt" dans le repo de POEM Mobile.

Vous pouvez lancer le fichier téléchargé sur AWS (bureau à distance).

Note

Si vous avez un problème avec le mot de passe, vous pouvez l'obtenir avec le fichier "private_key.pem".
Il suffit de cliquer sur le bouton "Obtenir le mot de passe" et d'importer le fichier "private_key.pem".

  • Si vous avez tout configuré vous pouvez lancer le serveur. Détails ici.
Attention

Si vous avez un problème de connection au serveur après l'avoir deployé. Vérifiez ici.
