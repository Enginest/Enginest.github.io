POEM

  • Ouvrez "PoemServer32.exe" dans /POEM

  • Cliquez sur "Server"

  • Cliquez sur "Settings..."

  • Dans "Root" mettre le chemin d'accès vers le dossier "TMS32".

  • Dans "Server" mettre l'ip et le port dans les champs correspondant.

  • Dans "Database" mettre "TMS32" dans le champs "Data source name".

  • Dans "Parameters" cliquez sur le bouton "Default".
