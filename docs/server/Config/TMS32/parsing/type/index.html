Types/Alias

Ca permet d'importer plus facilement des valeurs dans l'application.

Les différents types​

TypeDescriptionParameter
enumalias (0 => unavailable)number
dateimporte la date (2023-10-26)boolean
timeimporte le temps (12:43:29)boolean
datetimeimporte la date et le temps (2023-10-26 12:43:29)boolean
latitudeimporte la latitude (37.4226711)boolean
longitudeimporte la longitude (4.70897)boolean

Exemple​

Le type "enum" permet de sélectionner plus simplement. A la place de saisir 0, 1, 2, 3, etc... Cela fait correspondre les chiffres de la db (0, 1, 2, 3, etc...) à des alias (unavailable, available, planned, launched, etc...) qu'on peut sélectionner dans une liste déroulante.
